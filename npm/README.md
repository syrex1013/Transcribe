# transcribe-all (npm)

CLI audio transcription via Groq Whisper — npm distribution package.

This npm package downloads the correct pre-built binary for your OS and
architecture during `npm install`.

## Install

```bash
npm install -g transcribe-all
```

## Usage

```bash
transcribe recording.mp3 en
```
